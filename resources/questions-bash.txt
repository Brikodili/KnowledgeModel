man и info
Умеет пользоваться историей команд и поиском в истории
Понимает механизм прав на файлы в UNIX - чтение, запись, выполнение и группа, владелец, прочие
Умеет задавать набор файлов для команды с помощью маски "*", "?"
chmod, chown
mkdir
ls
cp, mv
ps ax
ln
touch
cat, more, less
su, sudo
tar, gzip, bzip
Использует команду ==alias==, добавил алиасы для своих команд (сокращений с часто используемыми ключами) в bash.rc
Использует ==screen== (или ==tmux==) для запуска на удалённом сервере (по ssh) команд, выполнение которых занимает длительное время
Понимает как действуют права на каталоги, знает что такое "sticky bit"
Умеет объединять команды в цепочки с помощью пайпов "|"
Пользуется многозадачностью в командной строке с помощью ==&==, ==jobs==, ==fg==, ==bg==
Применяет в аргументах команд конструкцию "{}", например для переменования или перемещения нескольких файлов
find
grep
sed, awk
wc
Умеет объединять команды в цепочки с помощью "&&", "||"
xargs
